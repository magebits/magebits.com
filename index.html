Business Goal

Integrated firewall with Magento helps you control traffic on your store independently of your HTTP server configuration.

Easy to use user interface provides a simple way to define rules per IP address.

Apart basic IP filter, built-in geolocation functionality allows to block traffic from unwanted countries.

Thanks to Firewall module you no longer have to configure each HTTP server manually.

This simplified management is a perfect replacement for .htaccess files.

Feature Highlights

IP filter

Define IPv4 rules for specific addresses to build your blacklist.

Geolocation filter

Set policy and restrict access to you store from selected countries.

Access log

Activity log shows allowed and denied requests.

Original source code is available in unencrypted and unobfuscated form.
